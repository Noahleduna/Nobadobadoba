Knapparnas funktion

Jag ändrade lite på tidigare kod och nu istället för ett if-kommando använder jag mig av nya mappar inuti min redan existerande casino app i visual studios för att ha spelen i för att ordna koden. Och skickar spelaren till en ny hemsida vid ett klick på spelknappen:

on:click={() => {  
            window.location.href = game.page;
        }}

där game.page kan vara t.ex. ‘/casino/blackjack’.

Jag lade också till så att knapparna blev större när man höll musen över dem samt sekade ner animationen:

    transition: transform 0.5s;
}
.game:hover{
    transform: scale(1.1);
